POST

/api/v1/resource

Create a new resource in the system.

201: Created { "status": true, "creator": "Hello Line", "data": { "title": "The MAX leveled hero will return!", "description": "Pahlawan level MAX telah ditambahkan!", "image": "https://cover.komiku.id/wp-content/uploads/2020/11/Manhwa-The-MAX-leveled-hero-will-return.png", "url": "https://komiku.id/manga/the-max-leveled-hero-will-return/" } }

DELETE

/api/v1/resource/:id

Delete a specific resource by ID.

200: Success { "status": true, "creator": "Hello Line", "data": { "id": "12345", "message": "Resource deleted successfully." } }
404: Not Found { "status": false, "message": "Resource with the given ID not found." }
500: Server Error { "status": false, "message": "Unable to delete the resource." }
